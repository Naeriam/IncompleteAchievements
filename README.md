# Incomplete Achievements (Addon for World of Warcraft) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A simple addon to have "Incomplete" option selected by default on achievements filter selector when you log in.

You can download this addon from:  https://www.curseforge.com/wow/addons/incomplete-achievements
